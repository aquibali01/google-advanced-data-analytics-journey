Context:

Objective:

• Project objective
• Business problem
• Success metrics
• Stakeholders
• Risks
• Data requirements
• Questions for discussion
• Timeline
• Deliverables
• Team responsibilities
• Feature engineering ideas
• ML model ideas
• Evaluation metrics
• Deployment considerations

Present the document as if it will be shared with executives and senior data scientists.
