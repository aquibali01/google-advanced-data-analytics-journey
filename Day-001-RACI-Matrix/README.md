# Day 1 — Understanding the RACI Matrix

## Course

Google Advanced Data Analytics Professional Certificate

---

## Today's Lesson

Today I learned how organizations define responsibilities during projects using the RACI Matrix.

R = Responsible

A = Accountable

C = Consulted

I = Informed

One task may have multiple Responsible people but usually only one Accountable person.

---

## Why It Matters

Many projects fail because responsibilities aren't clearly defined.

The RACI Matrix eliminates confusion and improves collaboration.

---

## Key Takeaways

• Multiple people can be Responsible

• Only one person should usually be Accountable

• Clear communication reduces project delays

• Data teams depend on clearly assigned ownership

---

## Reflection

Before today I assumed project success depended mostly on technical skills.

Now I realize communication and ownership are equally important.
